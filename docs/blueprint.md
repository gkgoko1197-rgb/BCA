# **App Name**: Employee Central Hub

## Core Features:

- User Authentication: Secure login system with distinct portals for users and admins.
- User Dashboard: Centralized hub for employees to view personal information and payslips.
- Dynamic Personal Info Updates: Allow users to edit personal information, with changes persisting using localStorage.
- Leave Request System: Enable employees to send leave requests to admins via a simple mail format and leave status display.
- Payslip Generation: Generate and display payslips with detailed employee and employer information, earnings, and deductions.
- Admin Dashboard: Dedicated admin interface to manage employee data, track attendance, and process leave requests.
- Attendance Calendar: Interactive calendar view for admins to monitor employee attendance, highlighting absences.

## Style Guidelines:

- Primary color: A calm blue (#64B5F6), evoking professionalism and trust.
- Background color: Light gray (#F5F5F5) to provide a clean and neutral backdrop.
- Accent color: Soft green (#A5D6A7) to highlight important actions and approvals.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and readable appearance.
- Use simple, clear icons to represent navigation items and actions.
- Employ a clean, organized layout with intuitive navigation and clear hierarchy.
- Incorporate subtle transitions and animations to enhance user experience and provide feedback.
